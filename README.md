# 🚀 Automação de Coleta de Dados - Oasis Tabacaria

Este projeto é uma ferramenta de **Inteligência Comercial** desenvolvida em Python para automatizar o levantamento de preços e estoque. Ele transforma uma busca manual demorada em uma planilha de Excel organizada, formatada e pronta para análise de mercado.

## 📈 Funcionalidades
- **Busca Automatizada:** Navega por todas as páginas de resultados de um termo pesquisado.
- **Inteligência de Dados:** Identifica automaticamente a marca, o preço e o link de cada produto.
- **Status de Estoque:** Verifica em tempo real se o item está disponível ou esgotado.
- **Excel Profissional:** Gera uma planilha com auto-ajuste de colunas, cabeçalhos destacados e links clicáveis.

## 🛠️ Tecnologias Utilizadas
- **Python** (Linguagem principal)
- **Selenium** (Navegação e Web Scraping)
- **Pandas & XlsxWriter** (Estruturação e formatação de dados)

## 💼 Valor para o Negócio
Como assistente, utilizei este projeto para reduzir o tempo de coleta de dados de concorrentes/fornecedores em **mais de 90%**, eliminando erros humanos de digitação e permitindo uma análise de preços muito mais ágil.
